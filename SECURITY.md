# Security

## Supported versions

We aim to support the latest release on the `main` branch. Use an up-to-date Node.js LTS for running the tool.

## Reporting a vulnerability

If you believe you have found a **security** issue in this project (e.g. unsafe file handling, dependency vulnerability with exploit path, credential exposure):

1. **Do not** open a public issue with exploit details.
2. Contact the maintainers through GitHub: use **Security advisories** on the repository ( **Security** → **Report a vulnerability** ) if the feature is enabled, or a private message channel if the maintainer has published one.

Please include a short description, steps to reproduce, and affected version or commit if known. We will work with you to assess and address the report responsibly.

## Responsible use of the scraper

This tool fetches and stores content from the web. Use it only in compliance with applicable laws and the terms of sites you access. The maintainers are not responsible for misuse.
